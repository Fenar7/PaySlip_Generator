# Getting Started with Slipwise One

Welcome to Slipwise One — the all-in-one business operations platform for modern enterprises.

## Quick Start

1. **Create an Organization** — After signing up, create your first organization. This is your workspace where all data (invoices, bills, payroll) lives.

2. **Invite Your Team** — Go to Settings → Users to invite team members. Assign roles (Owner, Admin, Member) or create custom roles with granular permissions.

3. **Configure Your Branding** — Upload your logo and set accent colors in Settings → Branding. These will appear on all generated documents.

4. **Set Up Customers & Vendors** — Add your customers and vendors before creating invoices or bills. You can import them via CSV.

5. **Create Your First Invoice** — Navigate to Docs → Invoices → New. Choose a template, fill in line items, and send.

## Platform Suites

| Suite | Purpose |
|-------|---------|
| **Docs** | Invoices, Quotes, Vouchers, Salary Slips |
| **Pay** | Bills, Payments, Payroll, Dunning |
| **Flow** | Banking, Reconciliation, Close |
| **Pixel** | PDF Studio, OCR, Templates |
| **Intel** | Analytics, Forecasting, KPIs |
| **Auth** | SSO, RBAC, Security |
