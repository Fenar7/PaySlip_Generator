## 📝 Technical Decisions

### Payment Gateway
- **Razorpay only** (no Stripe)
- Justification: India-first focus, lower fees, UPI support, enterprise features
- International cards supported via Razorpay

### API Authentication
- **Bearer token** (SHA-256 hash of API key) or **X-API-Key** header
- Rate limiting per plan tier
- Scope-based access control

### Enterprise Features
- **SAML 2.0** (no OIDC)
- **Multi-org** with role inheritance
- **Custom domains** validated via DNS A/CNAME/ALIAS records

### Infrastructure
- **Docker** for ECS deployment
- **GitHub Actions** for CI/CD
- **S3** for document storage
- **Redis** for caching and rate limiting
- **PostHog** for analytics
- **Sentry** for error tracking

### PWA Strategy
- Service worker with network-first cache strategy
- Offline page for graceful degradation
- Push notifications with VAPID keys
- Install prompt on iOS/Android browsers
